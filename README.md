# Overview
M3E player is a Material 3 Expressive UI designed Music player Webapp.

# It features
1. Material 3 expressive UI.
2. Light/Dark Mode Support.
3. Access Your Local Albums.
4. Repeat and Shuffle Support.
5. Choose your favorite songs.
6. Reprder tou playlist
7. Installs Locally and Works Offline.
8. Saves your Playlist data only (not your music files)

# Important
Due to some security reasons on many browser it's necessary to you to manually allow playlist to read by app.
